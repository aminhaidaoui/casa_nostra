# CASA NOSTRA — V1

Prima versione funzionante del regalo per Camilla.

## Come provarla
Apri `index.html` nel browser.

## Dove mettere i contenuti
- Foto: `assets/images/`
- Canzone: rinominala `canzone.mp3` e mettila in `assets/audio/`
- Testi, date, ricordi e messaggi: modifica `content.js`

## Cosa è già funzionante
- Intro narrativa
- Musica al primo click
- Layout responsive PC/telefono
- Giorni insieme dal 09/02/2026
- Saluto in base all'orario
- "Amin era qui"
- Galassia interattiva
- "Come ti senti oggi?"
- Porta segreta del 20/07/2026
- Timeline 5 mesi
- Ricordo casuale
- Easter egg "non aprire"
- Animazioni allo scroll

## Prossimo step
